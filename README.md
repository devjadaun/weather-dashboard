# 🌦 Weather Dashboard

A modern and interactive Weather Dashboard built using HTML, CSS, and JavaScript. This application provides real-time weather information, air quality data, weather forecasts, and beautiful weather-based animations for an enhanced user experience.

---

## 🚀 Live Features

### 🌍 Real-Time Weather
- Search weather by city name
- Displays current temperature
- Weather description
- Humidity levels
- Wind speed
- Atmospheric pressure
- Feels-like temperature

### 🌅 Sun Information
- Sunrise time
- Sunset time

### 🌫 Air Quality Index (AQI)
- Real-time AQI data
- Easy-to-understand AQI levels:
  - 😊 Good
  - 🙂 Fair
  - 😐 Moderate
  - 😷 Poor
  - 🚨 Very Poor

### 📅 5-Day Weather Forecast
- Daily weather forecast
- Weather icons
- Forecast temperatures

### 📊 Temperature Analytics
- Interactive temperature trend chart
- Built using Chart.js
- Visual temperature comparison

### 🎨 Dynamic Weather Backgrounds
The dashboard changes appearance according to weather conditions:

- ☀️ Clear Sky
- ☁️ Cloudy Weather
- 🌧 Rainy Weather
- ⛈ Thunderstorm
- 🌙 Night Mode

### ✨ Weather Animations
- Animated Clouds
- Falling Rain Effect
- Night Stars Animation
- Dynamic Sky Effects

### 📱 Responsive Design
- Desktop Friendly
- Tablet Friendly
- Mobile Friendly
- Smooth User Experience
